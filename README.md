# TeleSpamer
## Deployment
### Requirements
Docker and docker-compose needed
### Step By Step
#### 1
Run this in terminal
```bash
git clone https://github.com/SigachevAV/TeleSpamer.git
```
#### 2
Put your admin TelegramTag and Telegram BOT_TOKEN in docker-compose.yml
#### 3
Run this in terminal
```bash
docker compose build
```
#### 4
Run this in terminal
```bash
docker compose up -d
```
